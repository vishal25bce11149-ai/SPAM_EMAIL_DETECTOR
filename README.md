### Spam Email Detector

###### 

#### Project Overview

###### 

###### The Spam Email Detector is a machine learning-based project that classifies email messages as "Spam" or "Not Spam". It uses Natural Language Processing (NLP) techniques to analyze the content of emails and predict their category.

###### 

#### Features

###### 

* ###### Detects spam emails automatically
* ###### Uses Machine Learning (Naive Bayes)
* ###### Simple and beginner-friendly implementation
* ###### Real-time user input prediction

###### 

#### Technologies Used

###### 

* ###### Python
* ###### Pandas
* ###### Scikit-learn

###### 

#### How It Works

###### 

###### 1\. Email text is collected as input

###### 2\. Text is converted into numerical format using "CountVectorizer"

###### 3\. A "Naive Bayes model" is trained on labeled data

###### 4\. The model predicts whether the email is spam or not

###### 

#### How to Run

###### 

###### 1\. Install required libraries:

###### 

###### pip install pandas scikit-learn

###### 

###### 2\. Run the Python file:

###### 

###### python spam\_detector.py

###### 

###### 3\. Enter an email message and get prediction

###### 

#### Output Example

###### 

###### Input: "Win a free iPhone now!!!"

###### Output: "Spam"

###### 

###### Input: "Let's meet tomorrow"

###### Output: "Not Spam"
